# Node Image Hosting

A [Sails](http://sailsjs.org) application

## Install:

### Dependencies

+ [Node](http://nodejs.org/)
+ [Mongo](http://www.mongodb.org/)

```
git clone https://github.com/osn64/nodeimghost.git && cd nodeimghost
sudo npm install -g sails@0.10.x
npm install
sails lift
```
